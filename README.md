# mahadevaiahrashmi.github.io (v1 — backup)

Original version of Rashmi Mahadevaiah's personal portfolio website.

**Live (current):** https://mahadevaiahrashmi.github.io/ (deployed from `main`)

## Preview Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Open in browser
open http://localhost:3000
```

## Build for Production

```bash
npm run build
npm run preview
```

## Branches

- **`main`** — Active working branch (deployed to GitHub Pages)
- **`v1`** — This branch. Backup of the original website before enhancements.
